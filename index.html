Projects

InDevView

InDevView is a MERN stack web application designed to help aspiring or current developers practice their interview skills. Users can either create a room with a specific interview question or join a room via an entry code. After everyone is readied up, users in a room can chat, talk, or code in real-time with each other via WebSockets.

SplitUp

SplitUp is a clone of the full-stack web application, Splitwise. SplitUp is an expense-tracking application in which a user can keep track of the expenses between them and their friends. A user can check of all of their expenses and the details for each expense or check a friend's page and see what expense they and their friend are associated with.

GitHub Live Link

Event Horizon

Event Horizon is an action game built with vanilla JavaScript and HTML5 Canvas. Users use the mouse to aim and shoot and the keyboard to move and perform other actions. The goal of the game is survive as long as possible while getting as high of a score as possible.
